Deep midnight sky, 180 twinkling stars (the golden ones shimmer at a different rate), soft radial glow behind the harp, waveform in gold. The font is Cinzel Decorative for the title and EB Garamond italic throughout — classical and elegant.
The harp itself is a fully drawn SVG instrument with a curved column, arched neck, soundboard with a rosette engraving, a sun motif, laurel details at the joint, and tuning pins at every string. Click any string directly to pluck it. The strings animate on hit — they brighten, thicken briefly, then fade with a pluck ripple at the midpoint.
Keys A S D F G H J K L ; pluck strings 1–10. Z / X shift octave.
Voices:

Celestial — triangle wave + 2nd harmonic, clean and bright
Lyric — five-harmonic stack, warm and singing
Ancient — sawtooth through bandpass, nasal and resonant, sub-tone body
Ethereal — inharmonic bell partials (real bell ratios: 1, 2.756, 5.404…) + chorused duplicate

Glissandos — Ascend / Descend / Arpeggio / Cascade (both hands overlapping) / Olympus (slow stately ascent with double plucks on key strings). Keys 1–5.
Each string is attributed to one of the Nine Muses — the display shows which Muse governs the note you just played.
